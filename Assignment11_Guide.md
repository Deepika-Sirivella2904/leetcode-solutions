# Assignment 11: Personal Web Page Creation Guide

## Overview
This assignment requires creating personal web pages using three different platforms:
1. Google Sites
2. WordPress
3. Website Development API with predesigned templates
4. Host the created web pages

---

## Part 1: Google Sites Personal Web Page

### Step-by-Step Instructions:

1. **Access Google Sites**
   - Go to: https://sites.google.com
   - Sign in with your Google account
   - Click "+ Create" or "Blank" to start a new site

2. **Choose a Template**
   - Google Sites offers predesigned templates
   - Select "Personal", "Portfolio", or "Resume" template
   - Or start with a blank template for full control

3. **Personalize Your Site**
   - Add your name in the header
   - Include sections for:
     - About Me
     - Education
     - Skills
     - Projects
     - Contact Information
   - Upload a profile picture
   - Choose a color theme

4. **Add Content**
   - Use text boxes, images, and embedded content
   - Add links to your social media
   - Include your resume or portfolio items

5. **Publish Your Site**
   - Click "Publish" button
   - Choose your site URL (e.g., sites.google.com/view/yourname)
   - Set visibility to "Public" or "Anyone with link"
   - Click "Publish" again

---

## Part 2: WordPress Personal Web Page

### Option A: WordPress.com (Free & Easy)

1. **Sign Up**
   - Go to: https://wordpress.com
   - Click "Get Started"
   - Choose the free plan
   - Create your account

2. **Choose a Theme**
   - Browse free themes
   - Recommended themes: "Astra", "OceanWP", "Neve", "GeneratePress"
   - Select and activate your theme

3. **Customize Your Site**
   - Go to "Appearance" → "Customize"
   - Add your site title and tagline
   - Upload a logo or header image
   - Choose colors and fonts

4. **Create Pages**
   - Create "Home", "About", "Contact" pages
   - Use the block editor (Gutenberg) to add content
   - Add images, text, and other elements

5. **Add Plugins** (if available in your plan)
   - Contact Form 7 (for contact forms)
   - Elementor (for advanced page building)
   - Jetpack (for stats and security)

6. **Publish**
   - Your site is automatically published
   - Your URL will be: yourname.wordpress.com

### Option B: Self-Hosted WordPress (Advanced)

1. **Get Hosting**
   - Purchase hosting from providers like Bluehost, SiteGround, or Hostinger
   - Most offer one-click WordPress installation

2. **Install WordPress**
   - Use hosting control panel (cPanel)
   - Click "WordPress Installer" or similar
   - Follow the setup wizard

3. **Follow steps 2-6 from Option A**

## Part 3: Website Development API with Predesigned Templates

### Option A: Using GitHub Pages (Free)

1. **Create GitHub Account**
   - Go to: https://github.com
   - Sign up for a free account

2. **Choose a Template**
   - Browse GitHub Pages templates
   - Popular options:
     - Jekyll themes (Minimal, Cayman, Architect)
     - HTML5 UP templates
     - Bootstrap templates

3. **Create Your Repository**
   - Click "New repository"
   - Name it: `yourname.github.io`
   - Make it public
   - Initialize with README

4. **Upload Template Files**
   - Download a template or use GitHub Pages generator
   - Upload files to your repository
   - Edit index.html with your information

5. **Enable GitHub Pages**
   - Go to repository settings
   - Scroll to "GitHub Pages" section
   - Select source branch (usually main)
   - Your site will be available at: https://yourname.github.io

### Option B: Using Netlify (Free)

1. **Sign Up for Netlify**
   - Go to: https://netlify.com
   - Sign up with GitHub or email

2. **Create a Site**
   - Click "New site from Git"
   - Connect your GitHub account
   - Select your repository

3. **Deploy**
   - Netlify will automatically deploy your site
   - You'll get a URL like: https://yourname.netlify.app

### Option C: Using Carrd (Super Simple)

1. **Go to Carrd**
   - Visit: https://carrd.co
   - Click "Build your site"

2. **Choose Template**
   - Select from predesigned templates
   - Many free options available

3. **Customize**
   - Edit text, colors, and images
   - Add sections and content
   - Very easy drag-and-drop interface

4. **Publish**
   - Click "Publish"
   - Get your free carrd.co URL

## Part 4: Hosting Options

### Free Hosting Services:

1. **GitHub Pages**
   - Free static hosting
   - Custom domain support
   - Perfect for HTML/CSS/JS sites

2. **Netlify**
   - Free static hosting
   - Continuous deployment
   - SSL certificates included

3. **Vercel**
   - Free static hosting
   - Great for modern web apps
   - Automatic HTTPS

4. **Firebase Hosting**
   - Free tier available
   - Google infrastructure
   - Good for SPAs

5. **Cloudflare Pages**
   - Free static hosting
   - Fast CDN
   - Easy deployment

### Paid Hosting (for WordPress):

1. **Bluehost** - Popular for beginners
2. **SiteGround** - Good performance
3. **Hostinger** - Affordable
4. **WP Engine** - Premium WordPress hosting
