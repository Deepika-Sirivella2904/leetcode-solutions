# GitHub Repository Setup Guide

## Step-by-Step Instructions

### Step 1: Create GitHub Account
1. Go to: https://github.com
2. Click "Sign up" (top right corner)
3. Fill in:
   - Username (choose something unique)
   - Email address
   - Password
4. Verify your email address

### Step 2: Create New Repository
1. After logging in, click the **+** icon (top right corner)
2. Select **"New repository"**
3. Fill in repository details:
   - **Repository name**: `yourusername.github.io` 
     - Replace `yourusername` with your actual GitHub username
     - Example: if your username is `john123`, name it `john123.github.io`
   - **Description**: Personal portfolio website
   - **Public/Private**: Select **Public** (required for GitHub Pages)
   - **Initialize this repository**: Check these boxes:
     - ✅ Add a README file
     - ✅ Add .gitignore (select HTML if available)
4. Click **"Create repository"**

### Step 3: Upload Your HTML File
1. In your new repository, click **"Add file"** → **"Upload files"**
2. Drag and drop your `quick_start_template.html` file
3. If you want to rename it:
   - Change the filename to `index.html` (recommended)
4. Scroll down to the bottom
5. In the "Commit changes" box, type: "Add personal portfolio"
6. Click **"Commit changes"**

### Step 4: Enable GitHub Pages
1. In your repository, click on **Settings** (top menu)
2. Scroll down to the left sidebar
3. Click **"Pages"** (under "Code and automation")
4. Under "Build and deployment":
   - **Source**: Select "Deploy from a branch"
   - **Branch**: Select "main" (or "master")
   - Folder: Select "/ (root)"
5. Click **"Save"**

### Step 5: Wait for Deployment
- Wait 1-3 minutes for deployment to complete
- Refresh the page periodically
- You'll see a green checkmark when ready

### Step 6: Access Your Website
- Your website will be available at: `https://yourusername.github.io`
- Example: https://john123.github.io
- Open this URL in your browser to see your live website!

## Troubleshooting

### "Repository name already exists"
- This means someone else has that username
- Try adding numbers: `yourusername123.github.io`
- Or choose a different username

### Pages not showing
- Make sure repository is **Public**
- Wait at least 3-5 minutes
- Check the "Pages" tab for deployment status
- Try clearing browser cache

### Want to change the website content?
1. Go to your repository
2. Click on `index.html`
3. Click the pencil icon (edit)
4. Make your changes
5. Scroll down and click "Commit changes"
6. Changes will auto-update in 1-2 minutes

### Custom domain (optional)
If you want a custom domain like `yourname.com`:
1. Buy a domain from Namecheap, GoDaddy, etc.
2. In GitHub Pages settings, click "Add domain"
3. Follow DNS setup instructions
